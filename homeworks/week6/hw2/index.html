寫一點對這次作業的心得

因為已經落後進度了，所以挑戰題都先放置，只有先寫出基本題這樣，先來說說作業一，雖然之前也有在六角的 html 課程切過最終作業的版型，當時還拿了個優的回饋，但實際上自己一直都知道切板還有超多細節我沒注意到以及沒有搞懂的觀念，加上也沒切過 RWD 網站，所以這部分有時候對於區塊的安排還是有些模糊，有些時候雖然切出了和別人一樣的版型，但結構常常就是比別人還要「潮」，這大概是我目前最苦惱的其中一點了。

伴隨著上面這個問題延伸出來的就是 CSS 命名，自己是個有點命名潔癖的人，希望能以最符合語義的方式去命名，但總是搞不清楚該怎麼樣命名，反而更複雜，對明明也是感到頭很痛，尤其是單純容器的命名 wrapper。

另外對於 transition 的部分有些疑惑，為了讓滑鼠離開元素時也有動畫，因此使用了transition，但變成了載入網頁時就也會進行動畫，不知道這是否有辦法解決，再請 Huli 解惑了


作業二的部分，可能是因為進行作業之前為了學 RWD 有先看過 Udacity 的 RWD 課程，剛好有教到漢堡選單彈出的概念，所以這一開始看覺得還蠻容易的，甚至覺得應該跟作業一的順序相反才對？

實作之後，發現作業一的概念，在作業二的 RWD 主文版型非常好用，也藉此大概明白了 width、max-width 的差異以及該何時使用這樣。另外也學到了 em 和 rem 的差異，果然細節很多。

接著來說說這作業關鍵的拍手部分？

拍手部分，一開始遇到的問題是圖案，沒錯就是拍手圖案，這問題雖然因為自己就是設計科系的也會畫 icon，但我決定去學習之前在 slack 上同學說的 font icon 和 SVG 等作法，研究了一下之後，找到了幾個不錯的 font icon 網站，但是客製化操作上，比想像中麻煩，另外也確認了定位的方式要用 text。

回到隱藏的部分，原先的概念是要用位移的方式移出視窗之外，但是，不小心看到了一下 Huli 程式碼後，才發現，原來設定display 就好了，感覺自己真的是太死腦筋了。
